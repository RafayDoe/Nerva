# Nerva
A simple landing page built with React.js.

## Features
- Navbar
- Hero Section
- FollowUp Section
- Pricing Section
- Footer

## How to Install
1. Clone this repository

```sh
git clone https://github.com/RafayDoe/Nerva.git
```

2. Navigate to the project directory
3. Install dependencies

```sh 
npm install
```

Start the development server

```sh
npm run dev
```
